# CV Converter Agent

A Django REST + Next.js application for managing, converting, and formatting CVs with AI-powered features.

## Vector Search

Resume ↔ job description matching powered by vector similarity. Paste a job description and instantly find the best-matching candidates from your indexed CVs.

See [`backend/apps/vector_search/README.md`](backend/apps/vector_search/README.md) for full documentation, API reference, and setup instructions.
